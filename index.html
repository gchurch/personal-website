Superscalar Processor Simulator

Oct 2017 - Dec 2017, Summer 2018

In my 4th year I took the Advanced Computer Architecture unit. This unit involved us creating a simulator of a superscalar processor. I decided to create mine using C++. Features of the processor include:

  • A 5-stage pipeline (fetch, decode/issue, dispatch, execute, commit).
  • Reservation stations to store instructions that are waiting for operands to be computed.
  • Multiple execution units (load/store unit, ALUs, branch unit).
  • Out-of-order execution of instructions using a Reorder Buffer.
  • Register Renaming by use of a Register Allocation Table (RAT). This eliminates false data dependencies between instructions.
  • Simple branch prediction (predicts that every branch will be taken).

Cloud Computing Project

Oct 2018 - Dec 2018

In my 4th year I took the Cloud Computing unit. In the coursework for this unit, we had to create a scalable application that uses cloud computing resources. I created a simple website that allows you to upload images which are then displayed on the homepage. The web server was created with Node.js and Express. I used AWS resources to run the application. The web server is run using Elastic Beanstalk, which allows the web server to be automatically horizontally scalable. Images that are uploaded to the website are stored in an S3 bucket. DynamoDB is used for the database. CodePipeline is used for continuous delivery. I also used Apache Jmeter to load test the application.
